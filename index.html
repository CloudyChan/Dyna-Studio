Get started

To install Dyna Studio, you need to set up a Dyna Account and Developer Membership in the Dyna Homepage. Here's how: open the Dyna Developer Page in the Dyna Website by going to Accounts > Developer, click signup, and then follow the tutorial to create a Dyna Developer Account. You can read more about Dyna Developer Accounts in Dyna getting started guide for Dyna Studio. If you already have a Dyna Developer Account, sign in to your Developer account in the IDE by clicking on the avatar icon. After you sign in, install Dyna Studio by clicking the download button below. After the installion open the Dyna Studio App and click sign in with browser or sign in manually using the sign in page. You should see the Create, Open, Learn and Publish panels in the sidebar on the left side of your screen. Navigate to the download menu in Dyna Studio. There should be a few categories of programming languages and visual packages, select the packages you need and click download.
